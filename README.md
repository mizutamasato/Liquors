# 初めてのリカーズ

## サイト概要
### サイトテーマ
お酒に慣れていない人がみんなのお勧めのお酒を見たり、レビューできるレビューサイト

### テーマを選んだ理由
今はお酒が好きなのですが、初めて飲んだ時は美味しいとは感じませんでした。
周りでも最初のお酒が美味しくなくて、ずっと苦手意識を持っている人がいます。
そんな人でもお酒が簡単に楽しめるようなサイトがあればいいと考え、このテーマにしました。

### ターゲットユーザ
お酒に興味はあるけど慣れていない人

### 主な利用シーン
お酒を試したいとき、お勧めしたいお酒を投稿するとき
